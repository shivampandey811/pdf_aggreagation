# Charter Party PDF Automation

## Project Goals

- Automate extraction of Part I commercial terms and Part II clauses from input PDFs.
- Detect and highlight clause amendments: deleted text (strikethrough) and added/new text (green).
- Preserve clause numbering, titles, and layout fidelity.
- Provide an interactive Streamlit UI for upload, preview, and generation.
- Offer command line interface (CLI) for batch processing.
